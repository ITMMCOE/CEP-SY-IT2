Project Summary

Yarn Yantra is the digital storefront for a handmade crochet and yarn product brand. This project focuses on delivering a cozy, engaging, and mobile-first e-commerce experience while supporting a non-standard manual payment flow (UPI verification).

âœ¨ Key Features

Custom UPI Payment Flow: Secure checkout where customers pay via UPI, upload a screenshot, and orders are manually verified by the seller.

Admin Management Panel: Comprehensive back-end for the seller to manage products (stock, price, images), post blogs, and approve/reject pending orders.

Social \& Engagement Integration: Seamlessly displays the latest Instagram posts and provides a quick WhatsApp chat button for direct customer support.

Account Management: Users can log in/sign up, save addresses, and track their past orders.

Responsive Design: Built using React and Tailwind CSS for a consistent, high-quality experience across mobile and desktop devices.
